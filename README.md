## Simple name search 

using my name :  `Madonah`

```
Exercise 1. search Madonah.

OutPut : 
madonah = 1.0
madona = 0.0
madonna = 0.0
maddona = 0.0
maddoonna = 0.0
madonahhh = 0.0
madonayyy = 0.0
madonnauuu = 0.0
maddonattt = 0.0
maddoonnaiii = 0.0
```

> The confidence is always between 0 and 0.99999. So we got a positive score 1.0
